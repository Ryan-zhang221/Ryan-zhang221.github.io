搭建博客

需要的工具

1.node环境
2.npm
3.window10 系统
4.我们使用hexo来搭建博客

安装node环境和npm包管理工具

检查node环境 （node -v） 和npm包管理工具（npm -v），如果显示数字，格式为xx.xx.x就表示安装成功
如果使用npm安装较慢，可以使用国内的淘宝镜像来下载，以后就用cnpm代替npm来下载了：
(npm install -g cnpm --registry-https://registry.npm.taobao.org)

使用cnpm安装全局的hexo框架

(cnpm install -g hexo-cli)
使用(hexo -v)来验证安装是否成功

在当前文件夹下新建一个blog文件夹

以后博客所有的文件都存放在blog文件夹里，如果出现了错误，就直接删除该文件夹即可
使用(mkdir blog)即可新建blog文件夹
使用(cd blog)进入该文件夹中

使用hexo生成一个博客

命令行中键入(hexo init)来初始化一个博客，这里有人反映会有点慢，我自己下载挺快的，不知道是不是安装了git的原因
我们直接打开blog文件夹就可以看到自动生成的所有文件

启动博客

命令行键入(hexo s)
此时就会在命令行的4000端口启用
在浏览器中输入：http://localhost:4000 即可看到创建好的个人博客

写一篇文章

此时可以使用(hexo n "文章名称")来写一篇新的文章
也可以直接在 blog/source/_post 文件夹下新建一个.md文件，就是一篇新的文章，其中可以使用自己喜欢的编辑器写好，保存（这种方法简单）

将文章推送到博客中去

使用(hexo c)清空一下
使用(hexo g)生成一下
重新启动博客(hexo s)
这样就可以在 http://localhost:4000 中查看我们的新文章了

将博客部署到远端(github)上去

博客只能在本地使用当然不太好，所以我们需要将其部署到远端，我们可以通过域名来随时访问，这样博客的意义才体现出来
首先，创建一个自己的github账号
创建一个新仓库，仓库名必须是你的github用户名.github.io，如:ryan.github.io
装一个git部署的插件，键入(cnpm install --save hexo-deployer-git)
接下来设置_config.yml配置文件

配置_config.yml

键入(notepad _config.yml)会弹出一个记事本，找到最后一行，在deploy后面添加
    type: 'git'
    repo: https://github.com/你自己的仓库地址
    branch: master
保存即可

部署到远端即可

键入(hexo d)，可能要输入你的github账户名称和密码
接下来，直接输入https://你的仓库名称即可 就可查看自己的博客

增加或者修改文章

在blog/source/_posts/文件夹下找到要修改的文章进行修改或添加，修改完成后直接
(hexo c)(hexo g)(hexo d) 三次操作即可推送到远端

修改主题

如果现有主题不满意的话，可以在github上寻找好看的主题，克隆代码后放在blog/themes文件夹下
然后配置_config.yml文件，找到theme：修改为新主题的名称后保存，然后
(hexo c)(hexo g)(hexo d) 三次操作即可推送到远端
