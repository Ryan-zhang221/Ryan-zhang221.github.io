2.盒子模型、优先级、伪类

行内元素有哪些？块级元素有哪些？

  • 行内：a b span input
  • 块级：div ul li p h1 dl 独占一行的
  • 空：br hr

css的盒子模型

  • margin > border > padding > content

css选择符有哪些？哪些属性可以继承？优先级怎么算？

css选择符：

  • id #
  • class .
  • 标签 div、p
  • 相邻 ul+div
  • 子 ul>li
  • 后代 ul li
  • 通配 *
  • 属性 a[href=’xxx’]
  • 伪类 a:after a:hover

    css中哪些可以继承

  • color
  • font-size

    css选择器优先级问题（从高到低）

  • !important
  • 行内样式
  • id
  • class
  • 标签
  • 通配
  • 浏览器自定义或继承

css3新增伪类举例

  • :after 在元素内部最前添加内容
  • :before 在元素内部最后添加内容
  • :nth-child(n) 匹配父元素下指定子元素，在所有子元素中排序第n
  • :nth-last-child(n) 匹配父元素下指定子元素，在所有子元素中排序第n，从后向前数
  • :nth-child(odd)
  • :nth-child(even)
  • :nth-of-type(n) 匹配父元素下指定子元素，在同类子元素中排序第n
  • :first-of-type
